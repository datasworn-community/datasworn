# Notices

Datasworn was originally created by [rsek](https://github.com/rsek).

This repository is a clean-history community continuation under the
`datasworn-community` organization. It is not a mirror of upstream history.

The upstream source used as the reference for this restructure is
[`tbsvttr/datasworn`](https://github.com/tbsvttr/datasworn), which is itself a
community fork of `rsek/datasworn`.

Implementation files copied or adapted from upstream must be recorded in
`PROVENANCE.md` with their source paths and source commit.
